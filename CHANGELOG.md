# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-22

### Added

- Initial release
- 50 curated Three.js games with verified links
- Vue.js 3 with TypeScript implementation
- Modern glassmorphism UI design
- Animated particle background with mouse interaction
- Category filtering (Multiplayer, Racing, Horror, etc.)
- Search functionality by name, developer, category
- Free/Paid game filter
- Multiplayer game filter
- Grid/List view toggle
- Custom SVG icons for each game
- Responsive design for all screen sizes
- Game detail modal with play links
- Cloudflare Pages deployment support
- Custom 3D cube logo
- Gradient animations on titles
- Rotating border effects on cards
- Pulsing glow effects
- Sparkle background effects

### Technical Details

- Vue 3 Composition API with `<script setup>`
- TypeScript for type safety
- Vite for fast development and building
- CSS animations and transitions
- Canvas-based particle system
- Glassmorphism effects with backdrop-filter

### Performance

- Lazy loading for images
- Optimized bundle size
- Smooth animations at 60fps
